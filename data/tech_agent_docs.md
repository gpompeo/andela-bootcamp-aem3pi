# Tech Agent Documentation

## FAQ — Password reset

Employees can reset their password via the IT portal. Use the “Forgot Password” option, enter your company email, and follow the instructions sent to your inbox.

## Knowledge Base — VPN connectivity

To connect to the company VPN, install the VPN client from the IT portal. Use your company credentials and select the “Corporate” profile. Contact IT if you encounter repeated disconnections.

## SOP — Software license request

Submit a license request through the IT portal: Software > Request License. Include the software name, version, and business justification. Manager approval is required for paid licenses.

## Policy — Device usage

Company laptops and mobile devices must be used for work purposes only. Personal use should be minimal and compliant with security guidelines. Unauthorized software installations are prohibited.

## FAQ — Email quota

Each employee has a 50GB mailbox quota. When nearing the limit, delete old emails or archive them. IT will notify you if you exceed the quota.

## Knowledge Base — Wi-Fi troubleshooting

If you cannot connect to office Wi-Fi, restart your device, forget the network, and reconnect using your credentials. If issues persist, contact IT support.

## SOP — Opening a support ticket

Log into the IT portal and select “New Ticket.” Provide a detailed description of the issue, attach screenshots if possible, and categorize the problem. Tickets are prioritized based on severity.

## Policy — Remote access

Remote access is permitted only via VPN and company-approved devices. Access from public computers is prohibited. All remote sessions are logged for compliance.

## FAQ — Two-factor authentication

Two-factor authentication is required for all accounts. Use the company’s authenticator app or SMS codes. Contact IT if you lose access to your second factor.

## Knowledge Base — Printer setup

Install the printer driver from the IT portal. Connect to the printer via the network using its IP address. Default settings are managed centrally.

## SOP — Reporting phishing emails

Forward suspicious emails to security@company.com. Do not click links or open attachments. IT will investigate and advise next steps.

## Policy — Data backup

All company data must be stored on OneDrive or SharePoint. Local backups are prohibited. IT performs nightly backups of central servers.

## FAQ — Shared drive access

To gain access to a shared drive, submit a request via the IT portal. Manager approval is required. Access is granted within 2 business days.

## Knowledge Base — Collaboration tools

Teams is the primary collaboration tool. Use channels for projects, meetings for video calls, and SharePoint for document storage. Training materials are available in the IT portal.

## SOP — Hardware replacement

Submit a hardware replacement request in the IT portal. Include device serial number and issue description. IT will evaluate and replace if under warranty.

## Policy — Security patches

All devices must install security patches within 7 days of release. IT pushes updates automatically. Employees should not disable update services.

## FAQ — Mobile device management

Company mobile devices are managed via MDM. This ensures compliance with security policies. Personal devices may be enrolled voluntarily for email access.

## Knowledge Base — Ticket priority levels

Critical: system-wide outage. High: department-level issue. Medium: individual user issue. Low: minor inconvenience. IT responds according to SLA.

## SOP — Account deactivation

When an employee leaves, managers must submit an account deactivation request. IT disables accounts within 24 hours to protect company data.

## Policy — Cloud storage

Only company-approved cloud services (OneDrive, SharePoint) are permitted. Use of personal Dropbox or Google Drive accounts for work data is prohibited.

## FAQ — Video conferencing issues

If audio/video fails, check microphone and camera permissions. Restart Teams and reconnect. If problems persist, open a ticket with IT.

## Knowledge Base — Software catalog

The IT portal lists approved software. Employees can self-install free tools. Paid tools require manager approval and IT provisioning.

## SOP — Incident escalation

If an issue affects multiple users, escalate to IT immediately. Use the “Critical Incident” category in the ticketing system. IT will initiate response protocols.

## Policy — BYOD (Bring Your Own Device)

Employees may use personal devices for work if enrolled in MDM. Devices must meet minimum security standards. IT reserves the right to revoke access.

## FAQ — Locked account

If your account is locked after multiple failed login attempts, wait 15 minutes or contact IT to reset. Avoid repeated incorrect entries.

## Knowledge Base — Antivirus

All company devices run centrally managed antivirus. Employees should not disable or uninstall. Reports are monitored daily by IT.

## SOP — Software update

Employees must update software via the IT portal. IT provides instructions and schedules. Critical updates are mandatory within 48 hours.

## Policy — Internet usage

Internet access is monitored. Streaming and gaming during work hours are prohibited. Violations may result in disciplinary action.

## FAQ — Remote desktop

Employees can access their office desktop remotely via VPN and Remote Desktop Connection. Contact IT for setup instructions.

## Knowledge Base — Email forwarding

Automatic forwarding to external accounts is prohibited. Use Outlook rules for internal organization. IT audits forwarding rules quarterly.

## SOP — Device return

Departing employees must return all company devices. IT inspects and wipes devices before reuse. Managers coordinate collection.

## Policy — Encryption

All company laptops use full-disk encryption. Employees must not disable encryption. Lost devices must be reported immediately.

## FAQ — Software installation

Employees can install approved software from the IT portal. Unauthorized installations are blocked. Contact IT for exceptions.

## Knowledge Base — IT SLA

Critical issues: response within 1 hour. High: 4 hours. Medium: 1 business day. Low: 3 business days. SLA compliance is tracked.

## SOP — Network outage response

If the office network fails, IT initiates failover to backup systems. Employees should switch to mobile hotspots if necessary.

## Policy — Email retention

Emails are retained for 7 years per compliance. Employees should archive important emails. Deletion beyond retention policy is prohibited.

## FAQ — Shared calendar

Shared calendars are available in Outlook. Request access via IT portal. Managers can delegate permissions.

## Knowledge Base — Device inventory

IT maintains a central inventory of all devices. Employees can view assigned devices in the IT portal. Updates occur quarterly.

## SOP — Accessing logs

Managers can request system logs via IT. Submit a ticket specifying timeframe and system. IT provides logs within 5 business days.

## Policy — Remote monitoring

IT monitors device health remotely. Employees consent to monitoring by using company devices. Monitoring includes performance and security checks.

## FAQ — Conference room equipment

Conference rooms have standardized AV setups. Use HDMI or wireless casting. Contact IT if equipment malfunctions.

## Knowledge Base — Password policy

Passwords must be at least 12 characters, include uppercase, lowercase, numbers, and symbols. Passwords expire every 90 days.

## SOP — Firewall change request

Submit firewall change requests via IT portal. Include justification and affected systems. Security team reviews within 5 business days.

## Policy — Software licensing

All software must be licensed. Pirated or unlicensed software is prohibited. IT audits licenses annually.

## FAQ — File recovery

Deleted files can be recovered from OneDrive recycle bin within 30 days. Contact IT for older recoveries.

## Knowledge Base — IT training

Employees can access IT training modules in the Learning hub. Topics include security, collaboration tools, and troubleshooting basics.

## SOP — Device imaging

IT uses standard images for laptops. Imaging includes OS, drivers, and approved software. Employees receive devices pre-imaged.

## Policy — Remote work equipment

Employees working remotely must use company-approved equipment. IT provides laptops and peripherals. Personal equipment is not supported.

## FAQ — Browser compatibility

Company apps are optimized for Edge and Chrome. Other browsers may not be supported. Use approved browsers for best performance.

## Knowledge Base — Helpdesk hours

IT helpdesk operates 08:00–18:00 BRT, Monday–Friday. Critical incidents outside hours trigger on-call support.

## SOP — Escalating to vendor

If IT cannot resolve an issue, tickets are escalated to vendors. Employees are notified of timelines. Vendor SLAs vary.

## Policy — IT audits

IT conducts quarterly audits of systems, licenses, and compliance. Employees must cooperate with audit requests.

## FAQ — Shared mailbox

Shared mailboxes are available for teams. Request access via IT portal. Permissions are managed by IT.

## Knowledge Base — Cloud apps

Approved cloud apps include Office 365, Teams, SharePoint, and OneDrive. Other apps require IT review.

## SOP — Device disposal

Obsolete devices are securely wiped and recycled. IT coordinates disposal with certified vendors. Employees must not dispose of devices independently.

## Policy — Security awareness

Employees must complete annual security awareness training. Non-compliance may result in access restrictions.
